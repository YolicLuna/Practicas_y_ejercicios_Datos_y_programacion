# Proyectos de Análisis de Datos con Python

Repositorio de proyectos enfocados en análisis de datos, visualización y exploración utilizando Python.
Esta carpeta contiene prácticas orientadas a desarrollar habilidades en:
* Análisis exploratorio de datos (EDA).
* Visualización de datos.
* Series temporales.
* Interpretación de información.
* Uso de datasets reales. 

---

## Proyectos

### [Análisis de CO₂ per cápita](./Analisis_CO2)
Análisis exploratorio para estudiar la relación entre emisiones de CO₂ entre países.
* Comparación entre países.
* Cálculo de correlación.
* Visualización con gráficos de dispersión.
* Uso de widgets interactivos.

**Enfoque:** EDA + correlación + visualización

---

### [Análisis de Serie Temporal](./Analisis_serie_temporal)
Análisis de datos financieros a lo largo del tiempo utilizando técnicas de series temporales.
* Visualización de tendencias.
* Cálculo de medias móviles.
* Descomposición de la serie.
* Uso de datos desde APIs (yfinance).

**Enfoque:** análisis temporal + patrones + tendencias

---

### [Análisis del Índice de Felicidad](./Indice_de_felicidad)
Análisis comparativo del Índice de Felicidad Mundial a lo largo de múltiples años.
* Análisis de factores que influyen en la felicidad.
* Comparación de países por región.
* Tendencias temporales (2015-2019).
* Correlación entre variables económicas y felicidad.

**Enfoque:** análisis comparativo + tendencias + factores de felicidad

---

### [Análisis Superstore](./Superstore_analisis)
Análisis de datos de ventas de una tienda para examinar el rendimiento empresarial.
* Análisis de ventas y ganancias por región y categoría.
* Identificación de subcategorías más rentables.
* Impacto de descuentos en la ganancia.
* Análisis por segmento de cliente.

**Enfoque:** análisis de negocio + rentabilidad + patrones de venta

---

## Enfoque de la carpeta
Los proyectos de esta sección están orientados a:
* Trabajar con datos reales.
* Analizar relaciones entre variables.
* Identificar patrones y tendencias.
* Generar insights a partir de datos.
