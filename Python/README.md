# Ejercicios y Proyectos Python

Esta carpeta contiene proyectos y ejercicios prácticos de desarrollo en Python.

## Proyectos

### 1.- Tienda - Aplicación Streamlit

Una aplicación web interactiva desarrollada con **Streamlit** para gestionar una tienda de alimentos congelados. La aplicación permite:

- **Agregar productos**: Introducir nombre, precio y cantidad de productos
- **Calcular subtotales**: Calcula automáticamente el subtotal de cada producto
- **Visualizar carrito**: Muestra una tabla con los productos añadidos
- **Calcular total**: Calcula el monto total de la compra

**Tecnologías utilizadas:**
- Streamlit (interfaz web)
- Pandas (gestión de datos)

### 2.- Excel_Python - Transformación de Datos

Proyecto que demuestra la manipulación de archivos Excel con **Pandas**. Realiza transformaciones de datos de un archivo Excel, incluyendo:

- **Transformación de ID**: Genera IDs basados en las iniciales del nombre y número secuencial
- **Formateo de nombres**: Convierte nombres a formato título (Primera letra mayúscula)
- **Procesamiento de profesiones**: Convierte profesiones a mayúsculas
- **Cálculos**: Crea nuevas columnas con cálculos (ej: sueldo neto con descuentos)

**Tecnologías utilizadas:**
- Pandas (lectura y transformación de datos)
- OpenPyXL (manipulación de archivos Excel)

### 3.- Formulario - Aplicación GUI con PyQt5

Una aplicación de escritorio desarrollada con **PyQt5** para registrar y gestionar datos de empleados. Incluye:

- **Formulario de entrada**: Campos para nombre, apellido, profesión y sueldo
- **Tabla interactiva**: Visualización de los registros agregados
- **Cálculos automáticos**: Calcula el sueldo neto aplicando descuentos según rango salarial
- **Exportar a Excel**: Guarda los datos registrados en un archivo Excel

**Tecnologías utilizadas:**
- PyQt5 (interfaz gráfica de escritorio)
- Pandas (gestión de datos)
- OpenPyXL (exportación a Excel)
