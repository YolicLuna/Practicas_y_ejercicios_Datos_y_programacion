# Prácticas y Proyectos de Programación y Datos

Repositorio que contiene proyectos prácticos enfocados en el aprendizaje y aplicación de:
* Desarrollo con Python.
* Análisis de datos.
* Bases de datos.

---

## Propósito

Consolidar conocimientos mediante la construcción de proyectos reales que abordan:
* Desarrollo de aplicaciones (web y escritorio).
* Procesamiento y transformación de datos.
* Análisis exploratorio (EDA).
* Series temporales.

---

## Estructura del repositorio
Cada carpeta representa un área de estudio. Para más detalles, consulta el README correspondiente dentro de cada una.

---

## Tecnologías utilizadas
* **Python**: Pandas, NumPy, Streamlit, PyQt5, OpenPyXL.
* **Bases de datos**: MySQL.
* **Visualización**: Matplotlib, Seaborn.
* **Otros**: Jupyter Notebook.

---

## Carpetas principales

* **Python/**
  Proyectos de desarrollo en Python:
  * Aplicación web (Streamlit).
  * Procesamiento de Excel.
  * Aplicación de escritorio (PyQt5).
  * Gestor de tareas (Lista_de_Tareas).

---

* **Python_datos/**
  Proyectos de análisis de datos:
  * Análisis de emisiones de CO₂ (EDA y correlación).
  * Análisis de series temporales.

---

* **MySQL/**
  Ejercicios y consultas SQL enfocadas en análisis de datos
