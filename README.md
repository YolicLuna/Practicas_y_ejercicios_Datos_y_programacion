# Prácticas y Ejercicios de Programación y Datos

Este repositorio contiene una colección de ejercicios y proyectos de práctica enfocados en el aprendizaje de programación y análisis de datos.

## Propósito

El objetivo de este repositorio es consolidar y aplicar los conocimientos adquiridos en diferentes áreas de la programación y la ciencia de datos a través de ejercicios prácticos y casos de estudio.

## Estructura

Cada carpeta en este repositorio representa un área específica de estudio o un conjunto de ejercicios relacionados. Para más detalles sobre el contenido y los ejercicios de cada sección, consulta el README ubicado en su respectiva carpeta.

## Librerías y herramientas utilizadas

- Python (NumPy, Pandas, Streamlit)
- MySQL
- Y otras dependencias incluidas en los respectivos proyectos

---

Para información detallada sobre ejercicios específicos, consulta los README de cada carpeta.
